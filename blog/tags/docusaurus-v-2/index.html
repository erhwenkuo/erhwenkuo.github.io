靜態網站生成的好工具

November 7, 2021 · 3 min read
Erhwen Kuo
Lab-owner of Wistron DX Lab

Docusaurus是一個產生靜態網站生成器(static-site generator，雖然官方的Github中不以這個關鍵字稱它)。它使用了 React 技術，提供快速簡易的建置、佈署與維護文件型的網站。

什麼是靜態網站生成器​

它只是一種能將內容產生、佈署與維護靜態網頁(html)的工具，也就是說最終所有服務網站上的網頁都是 HTML 檔案。最近幾年這樣的工具並不少見, 以下列出大家討論比較多的幾種(按github的星星數來排序):

  • Next.js - https://nextjs.org/
  • Hugo - https://gohugo.io/
  • Gatsby - http://gatsbyjs.org
  • Jekyll - https://jekyllrb.com
  • Nuxt - https://nuxtjs.org/
  • Hexo - https://hexo.io/
  • Slate - https://slatedocs.github.io/slate/
  • Docusaurus - https://v2.docusaurus.io

經過一番執骰子看點數後，花了一些時間在Hubo與Docusaurus中左右為難。最後就選了有可愛icon的Docusaurus。

為什麼要使用靜態網站生成器來建置網站​

因為不需要去處理資料庫的設計，同時也不需要去寫網站伺服器端的後台執行程式(php, .net, node.js...)，再加上網站伺服器會作對靜態網頁作快取，頻寬使用上也會減少很多。

值得一提得是，目前靜態網站伺服器有很多免費的公有雲的服務可以用來佈署，也可以自訂自己的網址和有免費的SSL功能，省去了自架主機的工夫，所以也很省時省錢。

在企業內部使用了蠻方便的，透過一些CI/CD的pipeline整合與自動力，知識發佈與分享上便捷很多很多。
